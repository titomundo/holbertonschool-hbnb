# Holberton HBNB Project
This is Holberton HBnB. An educational project by Holberton School to teach students RESTful APIs, CRUD Operations, Software Architecture and Facade Design Pattern. This is achieved by building a simple AirBnB clone application using Python and the Flask framework.

### Requirements
- **User Management:** Users can register, update their profiles, and be identified as either regular users or administrators.
- **Place Management:** Users can list properties (places) they own, specifying details such as name, description, price, and location (latitude and longitude). Each place can also have a list of amenities.
- **Review Management:** Users can leave reviews for places they have visited, including a rating and a comment.
- **Amenity Management:** The application will manage amenities that can be associated with places.

### Layers
The app is divided in three main layers

- **Presentation Layer:** Includes the services and API through which users interact with the system.
- **Business Logic Layer:** Contains the models and the core logic of the application.
- **Persistence Layer:** Responsible for storing and retrieving data from the database.

### Structure
The project is divided in 4 parts, each focusing a different key aspect of the final product:
- `part1/` contains the design, technical documentation and diagrams for the application.
- `part2/` works on the Business Logic, Models, API endpoints and testing.
- `part3/` focuses on Authorization, Database connection and Persistent Storage.
- `part4/` creates a simple front-end app for users to interact with our application.
